So So Social Plug-In.
Powered by jQuery and YQL.

Plug-In Intro

The So So Social plug-in uses jQuery and YQL (Yahoo Query Language) to gather RSS feeds from your various social networks and sorts them into an "activity feed". Basically it creates a news feed out of your social networking activity.

Plug-In Features

  • No server side proxy needed! Thanks to YQL there is no need for any PHP, ASP, CFM, or whatever to Curl the RSS feeds.
  • Super easy to get set up and working.
  • Very light foot print.

Services Supported "Out of the Box"

  • Twitter
  • Facebook
  • Wordpress
  • Tumblr
  • Last.FM
  • Delicious
  • Flickr

I felt this was a sensible set of service to include with the plug-in. If there is a service not represented like posterous or virb or youtube...the code is open...extend it.
